# Security Policy

Thank you for helping keep Yukihana OS secure.

## Reporting a Vulnerability

Yukihana OS is currently an experimental educational operating system.

If you discover a security vulnerability that **does not contain sensitive information**, please open a GitHub issue describing the problem.

If the report contains sensitive information (such as an exploit that should not be publicly disclosed), please use GitHub's private vulnerability reporting feature if available. If it is not available, please contact the me privately before opening a public issue.

## Supported Versions

As the project is under active development, only the latest commit on the `master` branch is considered supported for security fixes.

Older revisions may not receive updates.
